# Simulación numérica de reacciones químicas con Julia

Este repositorio contiene el código y la documentación del proyecto introductorio
*"Simulación numérica de reacciones químicas y difusión"*, enfocado en la implementación
y comparación de métodos numéricos básicos (Euler explícito, Euler implícito y regla
trapezoidal) aplicados a sistemas de ecuaciones diferenciales ordinarias que modelan
reacciones químicas.

## Estructura
- `code/julia/`: Código fuente en Julia con los integradores temporales y scripts principales.
- `notebooks/`: Notebooks interactivos de **Pluto.jl** con explicaciones paso a paso.
- `results/`: Figuras y tablas generadas.
- `docs/`: Informe final y presentación.

## Requisitos
- Julia 1.9 o superior.
- Paquetes: `Plots`, `Pluto`, `LinearAlgebra`, `LaTeXStrings`.
